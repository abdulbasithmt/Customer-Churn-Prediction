# 📊 Customer Churn Prediction

## 📌 Objective
Predict whether a customer will churn using machine learning techniques.

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn

## 🔍 Process
- Data cleaning and preprocessing
- Feature encoding
- Model training using Logistic Regression
- Model evaluation

## 📊 Results
- Achieved ~80% accuracy
- Used confusion matrix and classification report for evaluation

## 💡 Insights
- Customers with higher charges are more likely to churn
- Long-term customers are less likely to leave

## 🚀 Conclusion
This model helps businesses identify potential churn customers and take preventive actions.
